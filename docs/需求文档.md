## 一、任务管理

| # | 功能点 | 状态 | 说明 |
|---|--------|------|------|
| 1.1 | 创建任务 | ✅ 已实现 | 输入任务名称，选择类型/紧急度/难度后提交 |
| 1.2 | 任务类型 | ✅ 已实现 | 作业 / 复习 / 兴趣 / 其他 |
| 1.3 | 紧急度 | ✅ 已实现 | 高 / 中 / 低 |
| 1.4 | 难度 | ✅ 已实现 | 易 / 中 / 难 |
| 1.5 | 状态流转 | ✅ 已实现 | 待开始 → 进行中 → 已完成，已完成可恢复为待开始 |
| 1.6 | 智能排序 | ✅ 已实现 | 优先级：状态（未完成在前）> 紧急度（高优先）> 难度（难优先）> 更新时间 |
| 1.7 | 任务建议 | ✅ 已实现 | 根据紧急度×难度（9种组合）自动生成一句行动建议 |
| 1.8 | 完成时鼓励 | ✅ 已实现 | 任务标为完成时，自动调用伴学话术在聊天区显示一句鼓励 |

| 1.10 | 编辑任务 | ❌ 未实现 | 后端 PATCH 支持改名/改属性，但前端没有编辑入口 |

## 二、番茄钟

## 三、伴学话术（短信陪伴）

| # | 功能点 | 状态 | 说明 |
|---|--------|------|------|
| 3.1 | 聊天对话 | ✅ 已实现 | 用户输入文字，返回关系人格对应的回复，显示在聊天气泡中 |
| 3.2 | 四种关系人格 | ✅ 已实现 | 兄弟（直爽激励）/ 闺蜜（温柔共情）/ 情侣（亲昵关怀）/ 学长（理性指导） |
| 3.3 | 关键词智能路由 | ✅ 已实现 | 消息含"搞定/完成/做完/好了/写完"→ 走完成鼓励池；含"烦/不想/累/拖/焦虑/慌"→ 走焦虑安抚池 |
| 3.4 | 点击角色轻互动 | ✅ 已实现 | 点击左侧角色区域，toast 弹出一句轻互动（如"拍肩：稳住"） |
| 3.5 | 回复内容 | ⚠️ 硬编码 | 每种人格每个场景 2~4 条预设文案，随机选取，非 AI 生成 |
